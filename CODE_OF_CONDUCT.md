# Code of Conduct

## Our Pledge

We are committed to providing a respectful, inclusive, and harassment-free community for everyone, regardless of background, identity, experience level, or perspective.

Krishi Sampark is built to support farmers, students, developers, researchers, and agriculture advisors. We expect all participants to help create a constructive and welcoming environment.

## Expected Behavior

Participants are expected to:

- Be respectful, patient, and constructive.
- Assume positive intent where reasonable.
- Focus feedback on ideas, code, documentation, and technical issues — not on people.
- Welcome different viewpoints, cultures, languages, and experience levels.
- Communicate clearly and professionally.
- Respect privacy and confidentiality.
- Give credit where appropriate.

## Unacceptable Behavior

Unacceptable behavior includes:

- Harassment, intimidation, insults, or personal attacks.
- Discrimination or exclusionary behavior.
- Trolling, deliberate disruption, or repeated off-topic comments.
- Doxxing or publishing private information without permission.
- Inappropriate sexual, violent, or offensive content.
- Abuse of project communication channels, issues, pull requests, or discussions.
- Misrepresentation of project ownership, affiliations, contributions, or results.

## Responsible Use

Because this project relates to agriculture advisory, participants should avoid making unsafe or unsupported claims about fertilizer, pesticide, disease treatment, financial outcomes, or field practices.

Contributions should clearly distinguish between:

- Implemented functionality
- Demo or sample functionality
- Roadmap or future work
- Expert-validated guidance
- AI-generated suggestions

## Enforcement

Project maintainers may review and take action on behavior that violates this Code of Conduct.

Possible actions include:

- Requesting clarification or edits
- Removing comments, issues, pull requests, or discussions
- Temporarily or permanently restricting participation
- Escalating serious concerns to the appropriate platform or authority when necessary

Enforcement should be fair, proportionate, and focused on protecting the project community.

## Reporting

If you experience or observe unacceptable behavior, please report it to the project maintainer.

For this capstone project, reports may be submitted through:

- GitHub issue or discussion, if appropriate
- Direct contact with the project maintainer
- Kaggle competition communication channels, where applicable

Please include relevant context such as links, screenshots, dates, and a short description of the concern.

## Scope

This Code of Conduct applies to all project spaces, including:

- Repository issues
- Pull requests
- Discussions
- Documentation comments
- Demo feedback
- Kaggle submission discussions
- Related project communication channels

It also applies when an individual is representing the project in public spaces.

## Attribution

This Code of Conduct is adapted for the Krishi Sampark capstone project and is intended to support a respectful, safe, and constructive open-source collaboration environment.
