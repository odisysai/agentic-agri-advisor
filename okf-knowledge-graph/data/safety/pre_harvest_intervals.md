# Pre-Harvest Intervals (PHI)

## Overview
PHI is the minimum number of days that must pass between the last pesticide application and crop harvest. Exceeding PHI can result in unsafe residue levels on food.

## Standard PHIs by Pesticide Type

| Pesticide Type | Typical PHI | Notes |
|---------------|-------------|-------|
| Copper-based fungicides | 7 days | Bordeaux mixture, copper sulfate |
| Sulfur-based fungicides | 3 days | Very short, safe for organic |
| Carbendazim | 14 days | Common fungicide for rust |
| Mancozeb | 14 days | Broad-spectrum fungicide |
| Chlorpyriphos | 21 days | Longer PHI, use with caution |
| Imidacloprid | 21 days | Systemic, longer residual |
| Pyrethroids (Lambda) | 14 days | Moderate PHI |
| Neem oil | 0 days | Safe to harvest same day |
| Biopesticides | 0 days | Bacillus thuringiensis, etc. |

## Factors Affecting PHI

1. **Rainfall** — heavy rain can reduce effective PHI
2. **Temperature** — higher temps may accelerate degradation
3. **Crop type** — leafy vegetables may have shorter PHIs
4. **Application method** — foliar spray vs soil drench
5. **Formulation** — EC, WP, SG formulations differ

## Compliance Notes

- PHI is legally enforceable in many countries
- Farmers must keep application records
- Post-harvest testing may verify compliance
- Organic certification has zero synthetic PHI

## When to Escalate

- If harvest is within PHI → advise waiting or suggest alternative treatment
- If multiple pesticides applied recently → use longest PHI
- If crop is for export → stricter residue limits apply
