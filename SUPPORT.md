# Support

## Getting Help

Use the following channels for help:

- Open a GitHub Discussion for design and usage questions.
- Open a GitHub Issue for reproducible bugs or feature requests.
- For security concerns, follow the private reporting process in `SECURITY.md`.

## What to Include

To help maintainers triage quickly, include:

- Environment details (OS, Python version, browser)
- Steps to reproduce
- Expected vs actual behavior
- Relevant logs or screenshots

## Response Expectations

This is an actively developed project. Response times vary by maintainer availability and issue severity.
